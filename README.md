# Node.js 

Projeto simples baseado em um tutorial de Node.js, utilizando os módulos `fs`, `http`, `path` e `express`.

## Instalação

1. Clone o repositório:
   
    git clone 
   
2. Navegue até o diretório:

    cd seu-repositorio
  
3. Instale as dependências:

    npm install
    
    
## Uso

Rode o projeto:
    
    npm run dev
